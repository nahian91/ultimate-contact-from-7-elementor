Integrates Contact Form 7 with Elementor using a custom widget for easy drag-and-drop functionality.

== Description ==

Ultimate Contact Form 7 for Elementor allows you to easily integrate Contact Form 7 forms into Elementor with a dedicated widget. This plugin adds a new Elementor widget that lets you select and display any Contact Form 7 form on your website, making it simple to create and manage forms within the Elementor interface.

Features:
- Select any existing Contact Form 7 form from a dropdown in Elementor.
- Easily display Contact Form 7 forms within Elementor layouts.
- Simple and intuitive setup.

== Installation ==

1. Upload the `ultimate-cf7-elementor` plugin folder to the `/wp-content/plugins/` directory, or install the plugin via the WordPress admin plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once activated, the "Contact Form 7" widget will be available in Elementor.

== Frequently Asked Questions ==

= Do I need Elementor and Contact Form 7 installed? =

Yes, this plugin requires both Elementor and Contact Form 7 to be installed and activated on your website.

= How do I use the widget? =

1. Edit a page using Elementor.
2. Drag and drop the "Contact Form 7" widget from the Elementor sidebar to your page.
3. Select your desired Contact Form 7 form from the dropdown list in the widget settings.

= Will this plugin work with other page builders? =

This plugin is specifically designed to work with Elementor. It will not work with other page builders.
